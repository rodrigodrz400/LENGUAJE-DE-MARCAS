Secciones HTML5 y agrupadores de contenido:

Para mejorar la estructura de las páginas y evitar el uso de otras etiquetas (div) HTML 5 añade nuevas etiquetas para clasificar el contenido por secciones y según su utilidad o importancia. Estas etiquetas son:
  • <header>- Sirve para insertar información como título, logo, enlaces a modo de introducción de la página.
  • <nav>- Sección con enlaces a otras páginas, esto es, los los menús de navegación.
  • <section>- Permite insertar una sección general dentro de un documento. Suele contener <h1>, <h2>,... para crear jerarquías. Puede anidarse.
  • <article>- Es un elemento atómico con el objetivo de que pueda reutilizarse (post de un foro, entrada de un blog, artículo de periódico, comentario de un usuario, etc.
  • <aside>- Es una sección relacionada indirectamente con el contenido. Son por ejemplo las barras laterales, publicidad, enlaces a otras webs.
  • <footer>- Pie de página, que suele contener autor, copyright, año, mapa del sitio, etc.
Los elementos clásicos para establecer agrupamientos (HTML 4.01 y XHTML) son los div y span:
  • <div>
    • - Sirve para agrupar varios elementos dentro de un bloque. Permite tratarlos como un conjunto asignándoles un identificador, una clase o un estilo.
  • <span>
    • - Parecido a un div pero para elementos en línea. Sobre todo se utiliza para asirgnar nombre, estilo o clase a un trozo de texto.
Utilizando algunos estilos vistos en clase, principalmente:
  • position: absolute; indica que vamos a establecer una posición fija para ese <div> en la ágina. El resto de los valores situarán el elemento de diferente forma.
  • left (izquierdo), right (derecho), bottom (inferior) y top (superior): estos valores, seguidos de un número de píxeles, porcentaje u otros valores, establecen la distancia que habrá entre el div y el borde respectivo.
  • width (ancho) y height (alto): indican la altura o anchura que tendrá el cuadro.

Actividad 11

Crearemos una página con unos párrafos de texto, y agregaremos un índice de contenidos (un enlace a cada párrafo), lo convertiremos en un bloque independiente mediante una etiqueta <div> e intentaremos colocarlo en un lateral de la página empleando los estilos. Para terminar, probaremos a modificar la apariencia de ese bloque mediante estilos.

Actividad 12

Tomando una página de ejemplo con abundante texto, probaremos a dividirla en dos columnas y posteriormente en tres. Probaremos a separar el interior de las columnas con diferentes tamaños, para dilucidar cuál es la que aporta mayor legibilidad al texto.

Actividad 13

Tomando como información la contenida en la página crea una página, utilizando las etiquetas de estructura de HTML5 que estimes convenientes. Intenta aplicar, al igual que lo harías con los div estilo para posicionar cada dato en su lugar. No hace falta incluir todo los datos, únicamente el menú principal, los artículos, una cabecera y un pie.
